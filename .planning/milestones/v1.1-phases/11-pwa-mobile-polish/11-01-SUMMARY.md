## Decisions Made

1. Build-hash substitution at serve time — `pwa.py` reads `sw.js` and replaces `__BUILD_HASH__` per request. Simpler than a Dockerfile `sed` step and consistent with how `compute_tailwind_css_path()` already derives the hash.

2. `pwa_router` placed first in `include_router` block — makes it clear the router owns root-level paths (`/manifest.json`, `/sw.js`) that aren't shadowed by the `/static` mount.

3. Non-GET bypass (`if (req.method !== 'GET') return`) is the first guard in the SW fetch handler — ensures HTMX mutation requests (POST/PUT/DELETE with CSRF tokens) always bypass the SW and reach the server intact (T-11-02).

4. Maskable icon safe zone: 10% padding per side → content in center 80%. Solid cream-50 (#FAF7F2 = RGB 250,247,242) fill in the corners (alpha=255). Satisfies Chrome Lighthouse "Maskable icon present" audit.
