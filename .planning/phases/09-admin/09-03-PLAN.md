## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| admin form -> credentials service | Plaintext key crosses here once; must be encrypted at rest and never echoed back |
| handler -> SDK provider | Decrypted key is used to authenticate; must stay in handler scope |
| handler -> template/log | Only last_four may cross; the decrypted key must not |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-09-10 | Information Disclosure | decrypted API key in template context | mitigate | Display query uses ApiCredential (last_four/model_name/is_enabled) only; set/probe handlers keep cred.key local; test asserts key absent from response (SEC-6) |
| T-09-11 | Information Disclosure | API key in logs | mitigate | structlog emits provider + status only on ADMIN_API_CREDENTIAL_SET / ADMIN_PROVIDER_TEST; never the key |
