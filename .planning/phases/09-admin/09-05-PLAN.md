<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser -> /admin/backups/{filename} | Untrusted filename param; must not escape /app/data/backups |
| run-now handler -> event loop | A sync long-running pg_dump must not block the loop / APScheduler |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-09-20 | Tampering | path traversal on backup download | mitigate | Strict regex FIRST, then Path.resolve().is_relative_to() containment, then is_file(); test_download_path_traversal (encoded + raw) |
| T-09-21 | Denial of Service | async run-now blocks the event loop / nightly scheduler | mitigate | Run-now is sync def (threadpool) per D-07; not async |
| T-09-22 | Information Disclosure | serving arbitrary file types / wrong content-type | mitigate | media_type fixed to gzip/.sql octet-stream; only regex-matched names served |
| T-09-23 | Tampering | CSRF on run-now | mitigate | Hidden X-CSRF-Token field + CSRFFormFieldShim |
| T-09-24 | Elevation of Privilege | non-admin download/run | mitigate | require_admin on list + download + run-now; non-admin 403 |
</threat_model>
