## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| developer → repo | A future contributor may add a template containing `|safe`, `hx-on:`, or `hx-vals='js:'` — bypassing CSP enforcement. The grep test in this plan is the policy enforcement point. |
| repo → production NGINX | README NGINX block is the authoritative artifact a sysadmin copies. If HSTS or `X-Forwarded-Proto` are absent, cookies break in production. The docs grep test prevents the README from drifting. |
