<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| every external request → app | All five middlewares form the trust boundary; misordering would allow CSP nonce-missing responses, CSRF bypass, or session leakage between requests |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-09-01 | Tampering | Middleware order error: CSP nonce missing on response (pitfall 13.4) | mitigate | RequestContextMiddleware added LAST (outermost) so it mints `csp_nonce` first on the request path; SecurityHeadersMiddleware reads it from `scope["state"]` on the response path. Plan 03's defensive `CSP_FALLBACK` is the second line of defense. (ASVS V13.2.4) |
| T-09-02 | Tampering | CSRF check running AFTER expensive session DB lookup wastes resources on attack traffic | mitigate | CSRFMiddleware added AFTER SessionMiddleware so on the request path CSRFMiddleware runs FIRST (outside) and fail-fasts with 403 before session DB lookup happens (RESEARCH §3 stack-order recommendation) |
| T-09-03 | Spoofing | contextvars leaking request_id across requests if RequestContextMiddleware is not outermost | mitigate | RequestContextMiddleware is verified outermost; `clear_contextvars()` called both at entry and in finally (pitfall 13.2 captured in Plan 02) |
| T-09-04 | Tampering | Future contributor adding `@app.on_event("startup")` instead of using lifespan (Starlette 1.0 deprecation) | mitigate | Module docstring + source-grep test verify `on_event` does not appear. Plan 10 ADR captures the rule. (RESEARCH §14 deprecation table) |
| T-09-05 | Information Disclosure | OpenAPI docs at `/docs` exposing `/debug/proxy` and the stub `/login` to anyone | mitigate | `docs_url=None, redoc_url=None` disables both docs pages in Phase 1; Phase 2 may re-enable behind `is_admin` |
| T-09-06 | Denial of Service | StaticFiles serving a misnamed file leaking server paths | accept | StaticFiles serves only `app/static/`; path traversal protected by Starlette's StaticFiles class (well-trodden code); content-hashed filenames per FOUND-12 prevent enumeration of unhashed assets |
| T-09-07 | Tampering | `app.state.limiter` not set before a 429 fires → exception handler crashes | mitigate | `register_rate_limiter(app)` is called BEFORE any router include or middleware add; the slowapi exception handler is registered at the same time |
</threat_model>
