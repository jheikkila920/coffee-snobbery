| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-07-02 | Spoofing | Brute force on /setup before first admin exists | mitigate | Same `5/15minutes` limit applied to /setup (D-17); Phase 2 adds `SELECT FOR UPDATE` on `app_settings.setup_completed` to prevent concurrent setup races (AUTH-02) |
