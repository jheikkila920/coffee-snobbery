      @router.get("/debug/proxy", response_model=DebugProxyResponse)
      async def debug_proxy(request: Request) -> DebugProxyResponse:

          trusted_proxy_ips = settings.TRUSTED_PROXY_IPS
          scheme = request.url.scheme
          trusted_list = {ip.strip() for ip in trusted_proxy_ips.split(",") if ip.strip()}

              client_host=client_host,

              headers_honored=headers_honored,
          )

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser → app templates | User-supplied data rendered in templates could carry XSS payloads if autoescape is off or `|safe` is used |
| NGINX config → operator | A wrong NGINX config (missing HSTS, wrong `X-Forwarded-*`) silently breaks production security; the README is the operator-facing artifact |
| browser → /debug/proxy | The endpoint is PUBLIC in Phase 1; reveals NGINX trust list to anyone (mitigated by it not being secret) |
