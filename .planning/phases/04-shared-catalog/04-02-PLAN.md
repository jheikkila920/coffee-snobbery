**I. `app/services/form_validation.py` (NEW)** — single module-level helper:

```
def errors_by_field(exc: ValidationError) -> dict[str, str]:
    """Pivot Pydantic ValidationError.errors() into {field_name: message} dict.

    Used by every Phase 4 catalog router POST handler to populate the
    inline-form-fragment errors context (SEC-06 + D-04 universal pattern).
    """
    out: dict[str, str] = {}
    for err in exc.errors():
        loc = err.get("loc", ())
        field = next((str(p) for p in loc if not isinstance(p, int)), "_form")
        out[field] = err.get("msg", "Invalid value")
    return out
```

Module docstring references SEC-06 + D-04. `__all__ = ["errors_by_field"]`.
