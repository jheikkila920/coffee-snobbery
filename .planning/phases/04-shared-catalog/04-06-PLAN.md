**B. `app/routers/equipment.py`** — clone the roasters router shape, drop `/list` (no autocomplete), drop `as_modal` form-field handling (no mini-modal).

`router = APIRouter(prefix="/equipment")`.

Endpoints:
1. `GET /` → list page (full page or `fragments/equipment_list.html` on HX-Request).
2. `GET /new` → empty form fragment.
3. `POST /` → form params `type_: str = Form(..., alias="type")`, `brand: str = Form(...)`, `model: str = Form(...)`, `notes: str = Form("")`. Validate via `EquipmentCreate(type=type_, brand=brand, model=model, notes=notes)`. On ValidationError → 200 + form fragment + errors. On success → row fragment.

Note the FastAPI Form alias trick: `type_: str = Form(..., alias="type")` so the HTML field name is `type` while the Python identifier is `type_` (avoids the Python builtin name shadowing).

4. `GET /{equipment_id}/edit` → form fragment pre-populated.
5. `POST /{equipment_id}` → update.
6. `POST /{equipment_id}/archive` → archive + return updated row fragment.
7. `GET /empty-form` → `fragments/empty.html` (Cancel path).
