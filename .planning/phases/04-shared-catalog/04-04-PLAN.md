Module docstring: "Roasters CRUD service — CAT-01. Sync Session per Phase 3 D-07; kwargs API per Phase 1 D-14 / Phase 3 D-08; audit events per Phase 1 D-14 taxonomy."

Imports: `from __future__ import annotations`, `structlog`, `sqlalchemy.{func, select, update}`, `sqlalchemy.orm.Session`, `app.events.{CATALOG_ROASTER_CREATED, CATALOG_ROASTER_UPDATED, CATALOG_ROASTER_ARCHIVED}`, `app.models.roaster.Roaster`.

`log = structlog.get_logger(__name__)`.

Functions (signatures only — implementation follows credentials.py pattern):

1. `def create_roaster(db: Session, *, name: str, location: str | None, website: str | None, notes: str, by_user_id: int) -> Roaster` — instantiate, `db.add`, `db.flush()` (populate id), `db.commit()`, `log.info(CATALOG_ROASTER_CREATED, roaster_id=roaster.id, user_id=by_user_id)`, return the row. Per PATTERNS, store `website` as `str` (HttpUrl→str conversion happens in the router via `str(form.website) if form.website else None`).

2. `def get_roaster(db: Session, *, roaster_id: int) -> Roaster | None` — `db.execute(select(Roaster).where(Roaster.id == roaster_id)).scalar_one_or_none()`.

3. `def list_roasters(db: Session, *, include_archived: bool = False) -> list[Roaster]` — `select(Roaster).order_by(Roaster.name)`; if `not include_archived`, filter `archived.is_(False)`.

4. `def update_roaster(db: Session, *, roaster_id: int, name: str, location: str | None, website: str | None, notes: str, by_user_id: int) -> Roaster` — `db.execute(update(Roaster).where(Roaster.id == roaster_id).values(name=name, location=location, website=website, notes=notes, updated_at=func.now()))`, `db.commit()`, then re-fetch the row + `log.info(CATALOG_ROASTER_UPDATED, ...)`.

5. `def archive_roaster(db: Session, *, roaster_id: int, by_user_id: int) -> None` — `db.execute(update(Roaster).where(Roaster.id == roaster_id).values(archived=True, updated_at=func.now()))`, `db.commit()`, `log.info(CATALOG_ROASTER_ARCHIVED, ...)`. Soft-delete only — Phase 4 ships archive-only per CONTEXT discretion.

6. `def search_by_prefix(db: Session, *, query: str, limit: int = 50) -> list[Roaster]` — D-13 autocomplete helper: `select(Roaster).where(Roaster.archived.is_(False)).where(Roaster.name.ilike(f"{query}%")).order_by(Roaster.name).limit(limit)`. CITEXT makes `ilike` case-insensitive natively; no `func.lower()` wrapper.

Audit-event kwarg names: use `user_id` (not `by_user_id` — Phase 1 D-14 alignment, see comment at `credentials.py:196-197`).
