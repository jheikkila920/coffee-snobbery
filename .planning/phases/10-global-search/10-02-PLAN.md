Acceptance: `test_brew_note_user_scoping` GREEN — User A response excludes User B's note, User B's includes it. |
| T-10-XSS | Tampering | highlight() helper (services/search.py) | mitigate | This plan IMPLEMENTS `markupsafe.escape()` on every text fragment + `Markup` composition with only the literal `<strong>` tag raw. No `|safe` in service or template. Acceptance: `test_highlight_xss_safe` GREEN — output contains `&lt;script&gt;`, no raw `<script>`. |
| T-10-SQLI | Tampering | ILIKE pattern construction | mitigate | `pattern = f"%{query}%"` is bound only via `Column.ilike(pattern)` (SQLAlchemy bound parameter) — never interpolated into raw `text()`. Acceptance: grep `text(f"` returns 0 in search.py; queries use `.ilike(pattern)`. |
